# DigitalCatan


## boot 

各コンポーネントの開発環境を用意した後に

    $ ./start_service

## issue name 

    [ctrl|view|hw|env][feature|bug] issue name

## branch name

    [feature|fix][-issue num]/branch-name
      or
    [feature|fix]/branch-name
    
branch-name は 簡潔な表現の英語でお願いします。

ex) `featrue-23/add-hyper-view-power`, `fix/re-connection`
